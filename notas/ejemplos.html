Colección de Ejemplos

1. El Toro Simple

Permutaciones: - σ = (1,2,3)(4,5,6) - ρ = (1,4)(2,5)(3,6)

Invariantes: - Género: 1 (toro) - Vértices: 2, Aristas: 3, Caras: 1 - Fronteras: 1

Código 
viz = RibbonGraphVisualizer('(1,2,3)(4,5,6)', '(1,4)(2,5)(3,6)')
viz.generar_secuencia_completa(prefix='toro_')

2. Vértice con Self-Loops

Permutaciones: - σ = (1,2,3,4) - ρ = (1,2)(3,4)

Invariantes: - Género: 0 (esfera) - Vértices: 1, Aristas: 2, Caras: 1 - Fronteras: 3

Código 
viz = RibbonGraphVisualizer('(1,2,3,4)', '(1,2)(3,4)')
viz.generar_secuencia_completa(prefix='selfloop_')

3. Triángulo (K₃)

Permutaciones: - σ = (1,2)(3,4)(5,6) - ρ = (1,3)(2,5)(4,6)

Invariantes: - Género: 0 (esfera) - Vértices: 3, Aristas: 3, Caras: 2 - Fronteras: 2

Código 
viz = RibbonGraphVisualizer('(1,2)(3,4)(5,6)', '(1,3)(2,5)(4,6)')
viz.generar_secuencia_completa(prefix='triangulo_')

4. Theta Graph

Permutaciones: - σ = (1,2,3)(4,5,6) - ρ = (1,6)(2,4)(3,5)

Invariantes: - Género: 1 (toro) - Vértices: 2, Aristas: 3, Caras: 1 - Fronteras: 1

Código 
viz = RibbonGraphVisualizer('(1,2,3)(4,5,6)', '(1,6)(2,4)(3,5)')
viz.generar_secuencia_completa(prefix='theta_')

Nota: A pesar de tener la misma estructura de σ que el ejemplo 1, el cambio en ρ produce una configuración diferente.

7. Dumbell (Dos Vértices, Una Arista)

La configuración más simple:

Permutaciones: - σ = (1)(2) - ρ = (1,2)

Invariantes: - Género: 0 (esfera) - Vértices: 2, Aristas: 1, Caras: 1 - Fronteras: 1

Código

viz.generar_secuencia_completa(prefix='dumbell_')

Patrones Observados

Género Cero (Esfera)

Ribbon graphs con g = 0: - Self-loops - Triángulo K₃ - Bouquet - Dumbell

Género Uno (Toro)

Ribbon graphs con g = 1: - Toro simple - Theta graph

Relación V-E

Para ribbon graphs en esfera (g=0): - Si V > E: tiende a tener múltiples fronteras - Si V = E: configuración balanceada - Si V < E: no puede existir (violaría χ)

Crear tus Propios Ejemplos

Plantilla general:

# Define tus permutaciones
mi_sigma = '(1,2,3)(...)'
mi_rho = '(1,?)(2,?)(3,?)...'

# Crear visualizador
viz = RibbonGraphVisualizer(mi_sigma, mi_rho)

# Ver invariantes
print(viz.invariantes())

# Generar visualizaciones
viz.generar_secuencia_completa(prefix='mi_ejemplo_')
